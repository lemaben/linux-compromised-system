# Step 8 - Advanced Track

Already comfortable with Linux? Push further.

## Advanced Objectives
- Explain why your chosen red herring was harmless
- Use logs to identify one clue that supports your findings
- Review command history for evidence of prior investigation activity
- List two remediation actions you would still take if this were a real server
- Find at least one hidden file unrelated to the attack

## Deliverable
Create your own short incident summary in `~/investigation/report.txt`.

Keep it concise:
- indicators found
- artifacts removed
- permissions fixed
- remaining risk
